## Exercises

### 1. Student Grade Manager

A simple program that:

* Accepts student marks.
* Calculates the average.
* Assigns a grade based on the average.

### 2. Number Analyzer

A program that analyzes a number by checking whether it is:

* Positive or Negative
* Even or Odd
* Prime or Not Prime
* Calculates its factorial

### 3. Contact Book

A menu-driven application that allows users to:

* Add contacts
* View all contacts
* Search for a contact
* Delete a contact

## Learning Outcomes

After completing this project, I can:


