# Object-Oriented Python & Advanced Features

This project is part of my **AI Engineer Dev Weekend Journey**. It focuses on Object-Oriented Programming (OOP) concepts and advanced Python features that are commonly used in AI and Machine Learning libraries.

## Objective

The objective of this project is to understand how Python organizes code using classes, inheritance, decorators, generators, iterators, and list comprehensions while building a small library management application.

## Topics Covered

* Classes and Objects
* Constructors (`__init__`)
* Attributes and Methods
* Inheritance
* Method Overriding
* `super()`
* List Comprehensions
* Decorators
* Generators
* Iterators

## Mini Project

A simple **Library Management System** demonstrating:

* A class hierarchy using inheritance
* Method overriding
* A custom decorator for logging
* A generator to iterate over book titles
* A custom iterator for countdown functionality
* List comprehensions for concise data processing

## Features

* Create library items using classes.
* Inherit common functionality from a parent class.
* Override methods in child classes.
* Use decorators to log function execution.
* Generate values lazily using generators.
* Build custom iterators.
* Write cleaner code with list comprehensions.

## How to Run

Run the main project:

```bash
python main.py
```

Run the list comprehension examples:

```bash
python comprehensions.py
```

## Learning Outcomes

After completing this project, I can:

* Implement inheritance and method overriding.
* Write Pythonic list comprehensions.
* Create and apply decorators.
* Understand the difference between iterators and generators.
* Build small object-oriented Python applications.
