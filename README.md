# AI Engineer Dev Weekend Journey

A structured, hands-on roadmap to becoming an **AI Engineer**, built through practical projects, clean code, and continuous learning.

Instead of only studying theory, every topic in this journey is implemented as a standalone, GitHub-ready project with documentation and notes.

---

## About This Journey

This repository documents my progress as I build the skills required for modern AI Engineering.

The focus is on understanding concepts deeply while creating practical projects that demonstrate each topic.

Every module is independent and includes:

- Clean Python implementations
- Practical exercises
- Well-structured code
- Revision notes
- GitHub-ready documentation

---

# Goals

- Build strong Python fundamentals
- Master Data Structures & Algorithms
- Learn software engineering best practices
- Understand concurrency and performance optimization
- Develop Machine Learning and Deep Learning skills
- Build Computer Vision and NLP applications
- Explore Generative AI and Large Language Models (LLMs)
- Learn MLOps and production AI deployment
